# Repositório de Exercícios de Modelagem de Banco de Dados

Este repositório é dedicado ao armazenamento e organização de exercícios, projetos e estudos práticos relacionados à **Modelagem de Banco de Dados**.

Aqui, você encontrará exemplos de modelos conceituais e lógicos, abordando diferentes cenários de negócio.

## Objetivo

O principal objetivo deste repositório é:

-   **Praticar:** Aplicar os conhecimentos teóricos na criação de modelos de dados eficientes e normalizados.
-   **Documentar:** Manter um registro organizado dos exercícios e suas soluções.
-   **Aprender:** Servir como um portfólio de estudos e uma fonte de consulta para diversos padrões de modelagem.

## Estrutura do Repositório

Os exercícios estão organizados por tópico ou por tipo de projeto. Cada pasta deve conter os artefatos relevantes para o exercício em questão.

-   `petshop/`: Modelagem para um sistema de um petshop.
-   `jardim-botanico/`: Modelagem para um sistema de um jardim botânico.
-   `plataforma-cursos/`: Modelagem para uma plataforma de cursos.
-   `normalizacao/`: Exercícios sobre normalização.
-   `normalizacao/shopping-behavior-preferences-study/`: Normalização para um dataset de como os compradores interagem com produtos ou lojas. 

### Dentro de cada pasta de exercício:

Geralmente, você encontrará os seguintes arquivos em cada subdiretório:

| Arquivo/Pasta             | Descrição                                                        |
| :------------------------ | :--------------------------------------------------------------- |
| `README.md`               | Detalhes específicos do exercício, requisitos e restrições.      |
| `diagrama-conceitual.png` | Imagem do Esquema Entidade-Relacionamento (DER).                 |
| `diagrama-logico.png`     | Imagem do esquema lógico, mostrando tabelas, atributos e chaves. |
| `diagrama-conceitual.xml` | Código XML para o Esquema Conceitual BrModelo.                   |
| `diagrama-logico.xml`     | Código XML para o Esquema Lógico BrModelo.                       |

## Ferramentas Utilizadas

Os modelos e diagramas neste repositório foram criados utilizando as seguintes ferramentas (ou similares):

-   **Modelagem:** BrModelo
-   **SGBD:** PostgreSQL
-   **Linguagem:** SQL (Structured Query Language)
