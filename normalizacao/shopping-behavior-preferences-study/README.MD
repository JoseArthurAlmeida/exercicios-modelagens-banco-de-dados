# Sobre o Dataset

## Acesso ao Dataset

[Shopping Behavior & Preferences Study](https://www.kaggle.com/datasets/ranaghulamnabi/shopping-behavior-and-preferences-study)

## Contexto:
Este Dataset captura como os compradores interagem com produtos ou lojas — o que eles pesquisam, navegam, adicionam aos carrinhos e compram. Os campos típicos incluem carimbo de data/hora, ID do usuário ou da sessão, itens visualizados, consultas de pesquisa, eventos do carrinho, compras e, possivelmente, canal ou dispositivo. A análise ajuda a revelar padrões de compra, produtos populares, gargalos de conversão e tendências por hora do dia ou sazonais. Útil para tarefas como análise de funil, personalização, modelagem de recomendações e estratégia de marketing.

## Distribuição de recursos:
Este Dataset rastreia ações dos compradores, como visualizações, pesquisas, adições ao carrinho e compras. A distribuição de recursos mostra a frequência com que cada ação ocorre e a popularidade do produto. Útil para analisar tendências e melhorar recomendações.
